# Chronos
Chronos - это умный трекер времени, который помогает планировать неделю, отслеживать реальные затраты времени и анализировать твой прогресс ежедневных задач.
Это не просто трекер времени. Это личный проводник в мире продуктивности, который помогает превратить хаос задач в стройную систему.
Больше не нужно держать расписание в голове или в десятках Excel-таблиц. Chronos объединяет планирование, учёт реального времени и аналитику в одном простом и
элегантном интерфейсе.
