# Chronos
Chronos - это умный трекер времени, который помогает планировать неделю, отслеживать реальные затраты времени и анализировать твой прогресс ежедневных задач.
Это не просто трекер. Это личный проводник в мире продуктивности, который помогает превратить хаос задач в стройную систему.
Больше не нужно держать расписание в голове или в десятках Excel-таблиц. Chronos объединяет планирование, учёт реального времени и аналитику в одном простом и
элегантном интерфейсе.

Приложение разделено на три больших раздела, каждый из которых имеет собственную функцию.

## Первый раздел "Расписание"
В этом разделе пользователь самостоятельно составляет своё расписание на неделю.
Каждый день представлен в виде отдельной карточки (таблички), которая содержит:
- название дня недели,
- актуальную дату,
- 8 строк для заполнения (время — задача).
Tакже присутствует блок «Важные события», где пользователь может сохранять ключевые события на определённую дату (функция в разработке).
### Как редактировать раписание
- Чтобы заполнить строку, дважды кликните по нужной строке - она перейдёт в режим редактирования.
- Чтобы сохранить изменения, также дважды кликните по строке после ввода текста.
- Чтобы добавить строчку, кликните на "+" внизу карточки

| Система автоматически выделяет редактируемую строку для удобства.
